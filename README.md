# WageAware - 打工人薪资计算器

实时追踪工作时长和收入，智能计算加班工资。纯前端应用，可直接在浏览器中运行。

## 特性

- ⏱️ **实时打卡** - 精确追踪工作时间和实时收益
- 📊 **智能统计** - 自动计算加班工资、平均时薪等指标
- 📅 **日历视图** - 查看历史工作记录
- 💾 **本地存储** - 所有数据保存在浏览器 LocalStorage 中
- 🎨 **现代界面** - 美观的深色主题设计
- 📱 **响应式** - 完美适配手机和电脑

## 快速开始

### 本地运行

无需安装任何依赖，直接打开 `index.html` 即可使用：

```bash
# 方式 1: 双击 index.html 文件在浏览器中打开

# 方式 2: 使用简单的 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### GitHub Pages 部署

1. 将代码推送到 GitHub 仓库
2. 进入仓库设置 → Pages
3. 选择部署源为 `main` 分支，根目录 `/`
4. 保存后等待部署完成
5. 访问提供的 URL 即可使用

## 技术栈

- **纯 HTML + CSS + JavaScript** - 无需构建工具
- **自定义 CSS** - 轻量级工具类样式系统
- **Chart.js** - 通过 CDN 引入，用于数据可视化
- **LocalStorage API** - 数据持久化

## 使用说明

### 首次使用

1. 打开应用后，会提示设置薪资信息
2. 填写月薪、工作制度、工作时间等信息
3. 保存后即可开始使用

### 打卡功能

- 点击"上班打卡"开始记录
- 实时显示已赚金额和平均时薪
- 点击"下班打卡"结束记录并保存

### 快捷记录

- **补卡** - 手动添加历史工作记录
- **年假** - 记录带薪年假（100% 工资）
- **病假** - 记录病假（默认 80% 工资）
- **事假** - 记录事假（无工资）

### 统计视图

- 查看周/月/年度统计数据
- 平均时薪、平均加班时长等指标
- 时薪趋势图表

## 数据说明

所有数据保存在浏览器的 LocalStorage 中，包括：

- `wageaware_settings` - 薪资配置
- `wageaware_current_session` - 当前打卡会话
- `wageaware_history` - 历史工作记录

**注意**: 清除浏览器数据会导致记录丢失，建议定期备份。

## 开发

这是一个纯前端项目，无需构建步骤。直接编辑源文件即可：

- 修改 `js/*.js` 文件以更改逻辑
- 修改 `index.html` 以调整结构和样式
